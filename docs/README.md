## Кратко о продукте

Веб-приложение (SaaS) для SMM-специалистов, агентств и брендов:

- подключение нескольких аккаунтов в VK и Telegram;
- создание постов (текст + фото/видео) с превью для каждой соцсети;
- планирование и календарь публикаций с учётом часовых поясов;
- надёжная очередь с ретраями и журналом статусов;
- базовая аналитика и биллинг по тарифам.

**Стек:** Node.js 20 + NestJS + PostgreSQL + Redis/BullMQ (backend), React 18 + TypeScript + Vite (frontend).
